## Setting Up

- Clone the project

```shell
git clone git@github.com:SiddhantTotade/PaperLess-Frontend.git
```

- Change directory to PaperLess-Frontend

```shell
cd clone PaperLess-Frontend
```

- Install the dependencies (Make sure that you've installed [pnpm](https://pnpm.io/installation))

```shell
pnpm install
```


- Run the development server

```shell
pnpm run dev
```
